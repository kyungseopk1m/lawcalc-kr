# 법령·출처 참조

본 문서는 현재 lawcalc-kr 가 다루는 계산 표면 — 판결금 이자, 지연손해금, 상속분 간이 계산, 소송비용 검토 — 의 참고 출처를 정리합니다. 현행 법령과 사건별 절차 이력의 직접 확인을 대체하지 않습니다.

## 독립성 명시

lawcalc-kr는 법원, 전자소송, 대법원 또는 그 산하기관이 배포·보증·감수한 프로그램이 아닙니다.
법원 공식 프로그램, 상표, 로고, 설치 파일과 무관한 독립 오픈소스 프로젝트입니다.

본 섹션이 이 무관성 명시의 단일 출처입니다. README와 다른 docs 파일에서 이 문구를 반복하지 않고 본 문서로 링크합니다.

## 현재 이율 가정

- 민사 법정이율: 다른 법령이나 약정이 적용되지 않는 한 연 5%.
- 상사 법정이율: 상법 적용 시 연 6%.
- 소촉법 판결 지연이율: 2019-06-01 부터 연 12%, 과거 이율은 버전 관리된 데이터로 표현.

위 가정은 버전 관리되는 데이터 파일과 골든 테스트로 뒷받침됩니다. 본 문서를 실행 가능한 정답으로 간주하지 마세요.

## 현재 상속 범위

- 1991-01-01 이후 사망한 피상속인만 지원합니다.
- 현재 엔진은 1~4순위 법정 상속인과 배우자 규칙을 지원합니다.
- 직계비속과 형제자매에 대한 1차 대습상속만 지원합니다.
- 2차 이상 대습, 직계존속 / 4촌 이내 방계 친족에 대한 대습은 명시적으로 거부합니다.

## 현재 소송비용 범위

- v0.3.0 소송비용 표면은 지원되는 민사·가사·행정·보전 및 관련 사건구분에 대해 인지대, 송달료, 변호사보수를 통합 산정합니다.
- 인지대는 「민사소송 등 인지법」 의 현행 4구간 누진표를 사용합니다 — 1,000원 최소액, 100원 절사, 항소/상고 배수, 지급명령/화해 감액, 전자소송 감액, 재심 메타데이터 포함.
- 송달료는 버전 관리되는 시기별 단가 슬라이스와 검증된 「재일 87-4」 횟수 매트릭스를 사용합니다. 공식 출처로 검증되지 않은 항목은 엔진이 silent 추정 대신 거부합니다.
- 변호사보수는 「변호사보수의 소송비용 산입에 관한 규칙」 의 현행 8구간 표를 사용합니다. 항소심·상고심은 불복 범위를 `caseValue` 로 명시한 독립 호출 패턴으로 모델링합니다.
- 분배는 균등 분배와 소가비례 분배를 정수 원 단위로 지원합니다. 잔여 정책은 결정적입니다 — floor 분배 후 남는 원은 첫 당사자부터 순서대로 1원씩 배정합니다.
- KLAC 적용 사건 범위 이슈는 자격 여부가 계산기 외부 사실에 달려 있기 때문에 비차단 경고로 노출됩니다.

## 자료 활용 정책

내부 기획 참고용으로는 로컬에 보관한 법원 매뉴얼을 분석에 활용할 수 있습니다. OSS 저장소에는 HWP 매뉴얼 본문, 스크린샷, 설치 파일 바이너리, 추출된 리소스, 복제된 UI 를 포함하지 않습니다.

저장소에 허용되는 것:

- 출처 명칭, 공식 URL, 짧은 인용
- 독립적으로 작성한 설명
- 골든 테스트용으로 작성한 계산 입력과 기대 출력
- 공식 공개 자료에 대한 링크

저장소에 허용되지 않는 것:

- 재배포된 HWP / MSI 파일
- 디컴파일 또는 추출된 프로그램 내부 자료
- 법원 로고, 스크린샷, UI 모방
- 매뉴얼 본문의 복사

## 릴리스 전 검증

법령·도메인 데이터를 변경하는 릴리스마다, 영향을 받는 행이나 규칙을 공식 출처와 대조 확인하고 다음을 기록합니다.

- 출처 제목과 URL
- 시행일
- 이율 값 또는 상속 규칙
- 확인 날짜
- 리뷰어 이니셜 또는 commit 참조
